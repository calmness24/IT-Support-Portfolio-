# Help Desk Ticket Simulation

## Ticket Number 
HD-001

## Issue Reported
User unable to log into Microsoft Outlook and continously receiving password prompts. 

## Environment
- Windows 10
- Microsoft Outlook / Microsoft 365

## Troubleshooting Steps Taken
1. Verified internet connectivity.
2. Confirmed correct username and password.
3. Restarted Outlook application.
4. Cleared cached credentials.
5. Tested Microsoft 365 web login.
6. Reconfigured Outlook profile.

## Root Cause
User password had expired.

## Resolution
Password reset completed successfully and Outlook access restored.
